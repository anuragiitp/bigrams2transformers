🧠 Universal Approximation Theorem
What it does:
A neural network with at least one hidden layer and a non-linear activation function (like ReLU, sigmoid, or tanh) can approximate any continuous function on a compact input domain, to any desired accuracy, if it has enough neurons.
f̂(x) = Σ cᵢ · σ(wᵢᵀx + bᵢ)
⚡ Why Activation Functions Matter
🔸 Non-linearity: Without them, network = linear model
🔸 Complex patterns: Enable learning of curves, shapes
🔸 Universal power: Make approximation possible
🔸 Common types: Sigmoid, ReLU, Tanh
⚠️ Key Caveat
🔸 Existence ≠ Trainability
🔸 Finding weights is hard
🔸 May need huge networks
🔸 Optimization challenges
💡 Bottom Line: Neural networks can theoretically learn any function, but activation functions are the key to unlocking this power!
